2.原理：
2.1.登录163邮箱，
2.2.逐级定位其他文件夹和12306文件夹
2.3.打开第一封邮件，
2.4.定位单次价格price
2.5.单次价格加到总价中
2.6.点击下一封，找不到下一封时，说明循环完毕退出循环
3.准备工作
3.1 将自己邮箱的名称和密码 填写到config.xlsx
3.2 在邮件中建立规则 将所有12306发送的邮件都放到该文件夹中，查看页面元素，记录该文件夹的ID，填写到
    driver.find_element_by_id('_mail_component_229_229').click()中
